# ¿Qué es un puntero?

Un *puntero*, es aquel variable que almacena la dirección de memoria de otra variable
ya declarada anteriormente.
# Declaracion

```CPP
int a = 0;
int *p;
// Variable point a direcction address of variable 'a'.
p = &a;
```

Forma que podemos declarar un puntero:

```CPP
#include <iostream>

int main(){
int*p
int* p;
int *p;
int * p;
}
```
